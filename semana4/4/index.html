¿Qué son peticiones GET?

Las peticiones GET son un tipo de solicitud que se utiliza en el protocolo HTTP para obtener información de un servidor. Cuando un navegador o una aplicación hace una petición GET, está solicitando datos desde una URL específica, como por ejemplo el contenido de una página web. Esta petición no modifica nada en el servidor, solo lee o consulta información.

¿Qué es fetch?

Fetch es una función de JavaScript utilizada para realizar solicitudes HTTP de forma asíncrona, permitiendo obtener o enviar datos a servidores web sin recargar la página. Se basa en promesas para manejar respuestas y errores de manera más clara y estructurada.

¿Qué son funciones asíncronas?

Las funciones asíncronas son un tipo especial de funciones que permiten ejecutar tareas de forma no bloqueante, es decir, sin detener el flujo del programa mientras esperan que una operación termine. En lenguajes como JavaScript, se declaran con la palabra clave async y suelen trabajar junto con await, que indica que se debe esperar el resultado de una operación antes de continuar.

¿Qué son promesas en JavaScript?

Son un mecanismo para manejar operaciones asincrónicas. Representan un valor que puede estar disponible ahora, en el futuro o nunca. Una promesa tiene tres estados: pendiente (pending), cumplida (fulfilled) o rechazada (rejected).>

¿Qué es await?

Es una palabra clave que se usa dentro de una función llamada async para pausar la ejecución hasta que una promesa se resuelva. Facilita escribir código asincrónico de forma más clara y similar al código síncronico

¿Qué es for-each?

Es una forma de recorrer todos los elementos de una colección (como un arreglo o lista) y ejecutar una acción para cada uno, sin usar un índice. Es común en lenguajes como JavaScript y hace el código más simple y legible.No permite detener el recorrido con break ni return como en otros bucles.
