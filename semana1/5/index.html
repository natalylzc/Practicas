¿Qué es JavaScript?

Es un lenguaje de programación dinámico y versátil que se utiliza para crear interactividad y funcionalidades avanzadas en las páginas web. Permite que los sitios web respondan a las acciones del usuario, como clics, movimientos del ratón o entrada de datos, haciendo que la experiencia en línea sea más atractiva y dinámica. Funciona junto a HTML (que da estructura) y CSS (que da estilo) para crear experiencias web completas, siendo fundamental para el desarrollo web y también usado en aplicaciones móviles y otros sistemas. JavaScript es fundamental para construir la web moderna e interactiva que conocemos hoy.

¿Qué es var?

Es una palabra clave en JavaScript que se usa para declarar una variable, es decir, una especie de contenedor con un nombre para alamacenar datos como números, texto o valores lógicos, y que puede reasignada; sin embargo, su uso es menos común en código moderno comparado con let y const.

¿Qué es let?

Es utilizada para declarar variables que tienen un alcance de bloque, lo que significa que solo son accesibles dentro del bloque de código donde se definieron (como dentro de llaves{}). A diferencia de var, las variables declaradas con let no se pueden redeclarar en el mismo ámbito, ofreciendo un control máes estricto sobre el código y ayudando a prevenir errores comunes.

¿Qué es const?

Se usa para declarar variables que no se pueden reasignar depués de su creación, manteniendo su valor fijo, similar a let en su alcance de bloque, y es preferible para valores que no cambiarán.

¿Qué es una función?

Son bloques de código reutilizables que ejecutan una tarea específica cuando son invocados. Puedes pasarles información (parámetros) y también pueden devolver un resultado, lo que las hace esenciales para estructurar y hacer más eficiente el desarrollo de aplicaciones.

¿Qué es un if?

Un if es una sentencia condicional en programación. Sirve para que tu programa tome decisiones. Le dices: "Si se cumple esta condición, haz esto". Si la condición no se cumple, esa parte del código no se ejecuta. Es la base para crear lógica y que un programa no sea siempre igual.
