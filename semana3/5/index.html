¿Qué es ::after y ::before?

Son pseudoelementos que permiten insertar contenido visual antes o después de un elemento HTML, sin necesidad de modificar el marcado original. Se utilizan comúnmente para añadir detalles decorativos, iconos, efectos visuales o textos auxiliares sin alterar la estructura del documento. Requieren la propiedad content para funcionar.

Top

Es una propiedad que afecta la posición vertical de un elemento posicionado. No afecta a los elementos no posicionados.

Bottom

Especifica la distancia a la que se mueve el borde inferior del elemento con respecto a su posición normal.

Left

Especifica la distancia desde el borde izquierdo del elemento padre (o del contenedor) hasta el borde izquierdo del propio elemento.

Right

Especifica la distancia desde el borde derecho del elemento padre o( del contenedor) hasta el borde derecho del propio elemento.

Position

La propiedad se utiliza para definir cómo se ubica un elemento en la página web, , usando valores como static, relative, absolute, fixed o sticky.

¿Qué es Fixed?

Es un valor de la propiedad position que hace que el elemento se mantenga en una posición fija en la ventana del navegador, sin moverse aunque se haga scroll. Siempre permanece visible en el mismo lugar de la pantalla. Se posiciona respecto al viewport, no al documento ni a otros elementos.

Absolute

Es un valor de la propiedad position que posiciona un elemento respecto al contendor más cercano que tenga una posición distinta a static

Relative

Es un valor de la propiedad position que permite mover un elemento desde su posición original, sin cambiar el espacio que ocupa en la página.

Transition

Se refiere a una propiedad de CSS, no a una etiqueta HTML, que permite crear animaciones de cambio gradual entre dos estados de un elemento, como pasar de un color de fondo a otro al hacer clic o pasar el ratón por encima.

Keyframes

Son una regla que permite definir los pasos intermedios de una animación CSS, controlando cómo cambian las propiedades de un elemento a lo largo del tiempo.

Z-index

Cuando dos elementos se superponen, el z-index determina qué elemento aparece encima del otro. El z-index El valor se puede asignar con un número entero o valores globales como heredar, inicial y unset. Cuando se usan enteros, el z-index asignado al índice más alto aparecerá delante de otros elementos

¿Qué es border-radius?

Redondea las esquinas del borde exterior de un elemento. Puedes establecer un solo radio para crear esquinas circulares o dos para crear esquinas elípticas.
