¿Qué son los ciclos?

Los ciclos en programación, tambien llamados bucles, son herramientas fundamentales que permiten la ejecución repetida de un conjunto de instrucciones. Son la base para automatizar tareas y procesar colecciones de datos de manera eficiente. Ya sea que necesites realizar una acción un número específico de veces (como con el ciclo for), o mientras una condición particular se mantenga, o hasta que se cumpla (como los ciclos white y do-white), los ciclos te ayudan a escribir código más conciso y potente, evitando la repetición manual de código y haciendo tus programas más dinámicos

¿Qué es for?

Es una estructura que repite un bloque de código un número determinado de veces, siendo muy útil para automatizar tareas y procesar listas de datos de forma eficiente. Su estructura permite definir fácilmente el inicio, el fin y el paso de la repetición.

¿Qué es while?

El ciclo while repite un bloque de código mientras una condición sea verdadera. A diferencia del for, no siempre sabes de antemano cuantás veces se ejecutará, solo que seguirá mientras la condición se cumpla.

¿Qué es recursividad?

Es una técnica de programación en la que una función se invoca a sí misma para resolver un problema, dividiéndolo en subproblemas más pequeños hasta alcanzar un caso base que detiene el proceso. Cuando creamos un método recursivo debemos tener en cuenta que este tiene que terminar por lo que dentro del método debemos asegurarnos de que no se está llamando a si mismo todo el rato, lo que quiere decir que el ciclo es finito.

Operador AND

El Operador AND (&&) es una herramienta lógica que se usa para verificar si múltiples condiciones se cumplen al mismo tiempo. Solo si todas las condiciones conectadas por el AND son verdaderas, el resultado de la operación será verdadero. Si alguna de ellas es falsa, el resultado general será falso

Operador OR

El operador OR (||) es una herramienta lógica que requiere que al menos una de las condiciones sea verdadera para que el resultado sea verdadero. Solo si todas las condiciones son falsas, el resultado será falso.
