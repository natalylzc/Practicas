¿Qué son arreglos?

Un arreglo, también conocido como array. Es una coleccion de elementos que se almacena de forma ordenada. Cada elemento dentro del arreglo tiene un número de identificación único, llamdo índice, que generalmente comienza en 0, permitiendo acceder a cada item de manera específica. Los elementos dentro de un arreglo pueden ser de diferentes tipos de datos, como números, textos o booleanos, y esta estructura es fundamental para organizar y manipular conjuntos de información de manera eficiente. Un array se representa comúnmente con corchetes [] para declararlo y contener sus elementos, aunque la sintaxis exacta puede variar según el lenguaje de programación.

¿Como se llena en Java?

Los arrays se pueden llenar usando su inicialización en la declaración para un conjunto fijo de valores, o usando el método Arrays.fill() de la clase java.util.Arrays para rellenar todo o una parte del array con un valor especificado. También puedes llenar un array de forma dinámica usando un bucle for para asignar valores elemento por elemento o generar valores aleatorios con `Math.random() `.
